🚀 الخطوات السريعة:

   1️⃣ Clone المشروع:
      git clone https://github.com/abdulr7man-samir/Store
      cd Store/fashion-store

   2️⃣ ثبتي وشغّلي محلياً:
      npm install
      npm run dev

   3️⃣ افتحي في المتصفح:
      http://localhost:3000

   4️⃣ أضيفي بيانات Supabase في .env

   5️⃣ انشري على Vercel:
      ✨ اذهبي إلى https://vercel.com
      ✨ اختاري "Import from GitHub"
      ✨ ابحثي عن "Store" repository
      ✨ اختاري "fashion-store" كـ Root Directory
      ✨ أضيفي Environment Variables
      ✨ اضغطي Deploy!
