🚀 الخطوات التالية (خمس دقائق فقط):

   STEP 1: اذهبي إلى Vercel
   ─────────────────────────
   https://vercel.com
   اضغطي "Sign Up With GitHub"

   STEP 2: اختاري Repository
   ──────────────────────────
   اضغطي "New Project"
   ابحثي عن "Store"
   اختاري Repository

   STEP 3: إعدادات المشروع
   ──────────────────────────
   Root Directory: fashion-store
   اضغطي "Continue"

   STEP 4: بيانات Supabase
   ──────────────────────────
   أضيفي Environment Variables:
   
   NEXT_PUBLIC_SUPABASE_URL = https://your-project.supabase.co
   NEXT_PUBLIC_SUPABASE_KEY = your_anon_key_here

   STEP 5: Deploy
   ──────────────
   اضغطي "Deploy"
   انتظري 2-3 دقائق
   ✨ خلصتِ!
